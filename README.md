# ThyName 『君の名は。』
a vim color scheme

## Just tired of all dark vim color scheme, so come out a light one.

## Install

just copy `./colors/thyname.vim` to your path `~/.vim/colors/thyname.vim`
than set colorscheme as `colorscheme thyname`
