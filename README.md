# ThyName 『君の名は。』
a vim color scheme

## Just tired of all dark vim color scheme, so come out a light one.

## Install

```sh
mkdir -p ~/.vim/colors
git clone https://github.com/rupertqin/ThyName.git ~/thyname
cp ~/thyname/colors/thyname.vim ~/.vim/colors/ 
```
